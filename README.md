# Buried City - Web Version v0.5

A web-based port of the Buried City game, built with React and TypeScript.

## Play Online

Play the game online at: [githubfrostpixel.github.io/buriedcity/](https://githubfrostpixel.github.io/buriedcity/)

## Installation and Run

### Prerequisites

- Node.js >= 18.0.0
- npm

### Setup

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
